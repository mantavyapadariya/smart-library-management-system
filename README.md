
# 📚 Smart Library Management System

A console-based **Library Management System** developed in **Python**. The project provides separate interfaces for **Admin** and **User**, allowing efficient management of books, users, book issuing, reservations, ratings, and fines. All data is stored using **JSON** files.

---

## ✨ Features

### 👨‍💼 Admin Features
- Admin Login
- Add New Books
- Update Book Details
- Delete Books
- View All Books
- Search Books
- View Users
- View Issued Books
- View Reservations
- View Ratings
- 
### 👤 User Features
- User Registration
- User Login
- Search Books
- Issue Books
- Return Books
- Reserve Books
- Rate Books

---

## 🛠️ Technologies Used

- Python
- JSON
- File Handling
- Logging

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/mantavyapadariya/smart-library-management-system.git
```

2. Open the project folder.

3. Run the program:

```bash
python main.py
```
